انیمیشن‌های ورود

این انیمیشن‌ها برای نمایش ورود عناصر به صفحه استفاده می‌شوند. مناسب برای عناصری که قرار است با تأخیر یا در پاسخ به تعامل کاربر ظاهر شوند.

Fade In

ظهور تدریجی

/* Usage */
.element {
animation: fadeIn 1.5s infinite alternate;
}
/* Keyframe Code */
@keyframes fadeIn {
from {
opacity: 0;
transform: translateY(30px);
}
to {
opacity: 1;
transform: translateY(0);
}
}

چرخش مداوم

animation: spin 3s linear infinite;
}
/* Keyframe Code */

}

animation-timing-function: ease-in;

100% {

انیمیشن‌های ویژه
